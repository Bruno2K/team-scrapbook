# Sons das reações

Coloque aqui os arquivos de áudio (MP3) que tocam quando o usuário reage a um post ou comentário.

**Nomes obrigatórios:**

| Reação   | Arquivo        |
|----------|----------------|
| Headshot | `headshot.mp3` |
| Heal     | `heal.mp3`     |
| Burn     | `burn.mp3`     |
| Backstab | `backstab.mp3` |

Depois de salvar os 4 arquivos nesta pasta, os sons serão reproduzidos ao clicar em cada botão de reação. Se um arquivo não existir, o app continua funcionando sem erro (apenas sem som).
